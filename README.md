# PowerShell Mastery Path

A free, structured, module-wise PowerShell training program — entry level to master level, with runnable code in every module and a hands-on project at the end of each level.

**Live site:** https://sigilipelli.github.io/powershell-mastery-path/

## Contents

- **Level 1 · Entry** — PowerShell foundations & the object pipeline
- **Level 2 · Intermediate** — idiomatic, testable, API-connected scripts
- **Level 3 · Advanced** — modules, DSC, security, performance
- **Level 4 · Master** — production automation, DevOps & publishing

## Local development

```bash
python3 -m venv .venv
.venv/bin/pip install mkdocs-material
.venv/bin/python -m mkdocs serve
```
